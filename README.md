# Société General SGT # Société General SGT

Société General SGT est une société spécialisée dans les prestations informatiques à destination de professionnels.

## Activité
- Audit technique et fonctionnel de logiciels
- Analyse de code et d’architecture applicative
- Correction de bugs
- Maintenance corrective et optimisation logicielle

## Types de projets
Les projets réalisés concernent principalement des logiciels métiers, applications web et outils internes.

## Confidentialité
Les projets clients sont couverts par des clauses contractuelles de confidentialité.
Les dépôts techniques sont privés et accessibles uniquement aux clients ou partenaires concernés.

## Informations légales
- Société : Société General SGT  
- Activité : Prestations informatiques  
- Localisation : Rouen, France
